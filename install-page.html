Install on Desktop

Follow these steps to install OpenTravel on your computer:

open-travel-psi.vercel.app
Install OpenTravel
Install this application on your computer
1

Open in Chrome, Edge, or other supported browser

Visit https://open-travel-psi.vercel.app/ in a supported browser like Chrome, Edge, or other Chromium-based browsers.

2

Look for the Install Prompt

If you see an install prompt in the address bar or as a banner, click Install.

3

Or use the Menu

Alternatively, click the three dots menu in the top-right corner, then select Install OpenTravel... or Apps > Install this site as an app.

4

Confirm Installation

Click Install in the confirmation dialog that appears.

After installation, you can launch OpenTravel directly from your home screen or app drawer.
